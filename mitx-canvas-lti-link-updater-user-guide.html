Overview

MITx links in your Canvas course point to a specific version of your MITx course. When a new version is created for a new semester, those links still point to the old one. This tool takes your Canvas course export and returns a copy with every MITx link pointing to the new version, plus a report of what changed.

Note: Run this once per semester, after the new MITx course version exists. If you're not sure it exists yet, email mitx-support@mit.edu.

Getting Your Two Export Files

Old Canvas course (the one with the MITx links):

  1. In Canvas, go to Settings → Export Course Content
  2. Select "Course" and click "Create Export"
  3. Download the file (ends in .imscc)

New MITx course:

  1. In MITx Studio, open the new semester's course
  2. Go to Tools → Export and click "Export Course Content"
  3. Download the file (ends in .tar.gz)

You don't need to rename or open either file — the tool identifies them automatically.

Using the Online Tool

This is the easiest option and works for courses of any size, including large ones.

  1. Open the tool in your browser (your ET can give you the link)
  2. Drag both files into the upload area, in any order, and click "Check Files"
  3. Review the confirmation screen — it shows which course your links point to now and where they'll point after the update. If anything looks unexpected, cancel and re-export the correct course. Nothing is changed until you confirm.
  4. Confirm. In Chrome or Edge you'll be asked where to save the updated file; in other browsers it downloads automatically. Then download the report if you want it.

Your files stay private. The online tool runs entirely in your browser — your course files are never uploaded to any server. For large courses, Chrome or Edge work best.

Running the Tool on Your Computer

If you're comfortable with a terminal, you can run the tool locally instead. This also works for any course size and needs no internet.

  1. Put both export files in a folder (any file names are fine)
  2. Open a terminal and run: 
    python cli.py /path/to/your/folder
  3. The updated file and report appear in that same folder

Note: This requires Python and a copy of the tool. Email mitx-support@mit.edu for help getting set up.

Importing Into Canvas

  1. In Canvas, open the new semester's course
  2. Go to Settings → Import Course Content
  3. Select "Canvas Course Export Package" and upload the _updated.imscc file
  4. After the import, click a few MITx links in Student View to confirm they open the new course's content

Understanding the Report

Status Meaning Action
MATCHED Link found in the new MITx course and updated None
MISSING Link points to content not found in the new MITx course; left unchanged Update or remove the link in Canvas manually
NEW_ONLY The new MITx course has content with no Canvas link Optional — add a link if students should reach it from Canvas

Note: The MITx tool configuration link (just https://lms.mitx.mit.edu) always shows as MISSING. That's normal — it isn't a content link.

Troubleshooting & Additional Information

FAQs

  • "No LTI links found"
    • You probably exported the new Canvas course instead of the old one — the MITx links live in last semester's course
  • Many links show as MISSING
    • Usually the wrong MITx course was exported. Double-check the new course export and try again
  • The tool warns the files look like the same course version
    • One of the exports came from the wrong place — re-export the old Canvas course and the new MITx course
  • Links still point to the old course after importing
    • The original file was imported instead of the one ending in _updated.imscc — re-import the updated file

For anything else, email mitx-support@mit.edu with a short description and, if you have it, the report from the tool.
